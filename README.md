# Tracee Solar Mini-Grid Project

This repository contains the strategic and technical preparation materials for the **Rural Utility Infrastructure Development** meeting in Nigeria.

## 🚀 Presentation Materials
- **[Interactive Presentation Dashboard](index.html)**: A modern web-based dashboard for live client presentations. Includes:
  - **Savings Calculator**: Real-time ROI modeling based on diesel costs.
  - **Architecture Diagrams**: Professional SVG visualizations of power flow and distribution.
  - **Discovery Checklist**: Guided data capture for site bankability.
- **[Client Meeting Preparation (Markdown)](client_meeting_prep.md)**: The internal strategy guide and detailed speaker notes.
- **[Site Audit Briefing Note](site_audit_briefing.md)**: A one-page explainer for the 48-hour technical audit.
- **[Meeting Minutes & Decisions Template](meeting_minutes_template.md)**: A structured form to capture client data and decisions during the meeting.

## 📊 Technical Assets (SVGs)
- `minigrid_diagram.svg`: System architecture and energy flow.
- `distribution_layout.svg`: High-voltage transmission and smart metering layout.
- `capex_composition.svg`: Breakdown of capital expenditure.
- `capital_structure.svg`: Debt vs. Equity funding ratios.
